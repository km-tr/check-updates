## コミット詳細

### [`f2cbfbe`](https://github.com/obra/superpowers/commit/f2cbfbefebbfef77321e4c9abc9e949826bea9d7) — Release v5.1.0 (#1468)
**2026-05-04 15:05 PDT**

マージコミット。主な内容は以下の通り：

**worktreeスキル全面書き直し（PRI-974）**
- `using-git-worktrees`: `GIT_DIR != GIT_COMMON` によるサンドボックス検出を Step 0 に追加。ネイティブツール（EnterWorktree 等）を優先し、なければ `git worktree add` にフォールバックする detect-and-defer 方式を採用。TDD で 50/50 の合格率を確認。
- `finishing-a-development-branch`: 環境検出・プロベナンスベースのクリーンアップ（`.worktrees/` のみ削除）・バグ #940/#999/#238 修正・stale worktree pruning を追加。
- 5ハーネス（Claude Code・Codex CLI・Codex App・Gemini・Cursor）でのクロスプラットフォーム検証完了。

**レガシー機能削除**
- `/brainstorm`・`/execute-plan`・`/write-plan` スラッシュコマンドを削除（`commands/` 配下3ファイル）。
- 全スキルから Integration セクションを削除。
- `agents/code-reviewer.md` を削除し、`requesting-code-review` スキル内の Task ディスパッチテンプレートに統合。

**バグ修正・最適化**
- OpenCode の `getBootstrapContent()` にモジュールレベルキャッシュを追加し、エージェントループのステップごとの不要なファイル I/O を解消（#1202）。
- SDD が3タスクごとに一時停止する問題を修正（`requesting-code-review` の記述漏れが原因）。
- Cursor Windows 環境での SessionStart フックを `run-hook.cmd` 経由に修正。
- brainstorm サーバーの owner-PID 誤判定（EPERM を死亡と誤解）を修正（#879）。
- brainstorm サーバーのセッションディレクトリを `content/`（HTTP公開）と `state/`（非公開）に分離（セキュリティ改善）。

**ドキュメント・テスト**
- Gemini CLI のサブエージェントディスパッチマッピングを追加（`gemini-tools.md`）。
- Copilot CLI のプラットフォーム検出と `sessionStart` フック出力形式を追加。
- SDD 統合テストの3バグを修正（パス解決・SIGPIPE・`--plugin-dir` 未指定）。
- `test-requesting-code-review.sh` 新設（SQL injection 等の実バグ植え付けテスト）。
- `test-worktree-native-preference.sh` 新設（Step 1a TDD ゲートテスト）。
- OpenCode ブートストラップキャッシュの15項目の回帰テスト追加。
- Factory Droid・Codex・quickstart のインストールドキュメントを更新。
